# Resume Relevance Scoring System

A web-based Resume Relevance Scoring System developed using Natural Language Processing (NLP) techniques.

## Project Overview

This system analyzes resumes and compares them with job descriptions using TF-IDF and cosine similarity. It generates relevance scores, ATS feedback, matched skills, and missing skills to help users improve resume-job alignment.

## Features

- Upload PDF and DOCX resumes
- Compare resumes with job descriptions
- TF-IDF vectorization
- Cosine similarity scoring
- ATS-style keyword analysis
- Matched and missing skills detection
- Interactive dashboard
- Streamlit-based user interface

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- NLTK
- Pandas

## Installation

```bash
pip install -r requirements.txt
```

## Run Application

```bash
streamlit run app.py
```
